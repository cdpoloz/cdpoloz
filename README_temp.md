## Featured Projects

### cpz-mvvm-processing-controls

Strict MVVM UI framework for Processing, designed for real-time environments and explicit control over behavior.

- Fully decoupled input system
- Explicit composition model (no hidden binding)
- Sketch-owned theming
- Clear separation between View, ViewModel, Style, and Renderer

https://github.com/cdpoloz/cpz-mvvm-processing-controls

---

### cpz-mvvm-processing-template

Reference project showing how to integrate the MVVM controls framework into a Processing sketch.

- Input system integration
- Theme management
- Example UI composition and behavior wiring

https://github.com/cdpoloz/cpz-mvvm-processing-template

---

### cpz-utils

Reusable Java utility library with no external dependencies.

- Time utilities (Timer, FixedStepTimer, Stopwatch)
- Color utilities
- Noise abstractions for simulation and animation

Designed to be framework-agnostic and shared across projects.

https://github.com/cdpoloz/cpz-utils
